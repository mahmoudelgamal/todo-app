# todo-app
## How to setup
1. clone the repo 
```shell
git clone https://github.com/mahmoudelgamal/todo-app.git
```
### FE
1. install dependencies
```shell
cd .\FE\
npm install
```

## How to start
```shell
npm start
```
will be open in port 8080 [http://localhost:8080](http://localhost:8080)
### Server
1. install dependencies
```shell
cd .\BE\
npm install
```

## How to start
```shell
npm start
```
will be use in port 3000
